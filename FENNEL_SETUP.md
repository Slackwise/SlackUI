# Fennel Setup Instructions for Windows

1. Download fennel.exe from the official Fennel releases:
   https://github.com/bakpakin/Fennel/releases

2. Place fennel.exe in a directory included in your PATH (e.g., C:\Users\YourName\bin or C:\Windows).

3. Verify installation by running:
   fennel --version

4. To compile your Fennel files, use:
   fennel --compile Init.fnl
